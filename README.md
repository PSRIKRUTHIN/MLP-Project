# MLP-Project

# Comment Category Prediction using Machine Learning  


This repository contains a complete end-to-end Machine Learning project developed as part of the **IIT Madras BS Diploma in Data Science** program.

The project focuses on predicting comment categories using data preprocessing, exploratory data analysis (EDA), feature engineering, and machine learning classification models.

---

## Project Overview

The objective of this project is to build a machine learning pipeline capable of classifying comments into different categories based on structured input features.

The project demonstrates practical implementation of:

- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Building
- Model Evaluation

This notebook follows a structured workflow similar to real-world industry data science projects.

---

## Dataset Features

The dataset contains multiple numerical and categorical features including:

- Upvotes & Downvotes
- Emoticon Features
- Gender & Race Information
- Conditional Features (`if_1`, `if_2`)
- Comment-related attributes

The project includes handling of missing values, outliers, and feature distributions before model training.

## Exploratory Data Analysis (EDA)

Performed detailed EDA using:

- Boxplots
- Heatmaps
- Feature Distribution Plots
- Statistical Summaries

### Key Insights

- Several emoticon features were highly sparse
- Upvote-related features contained outliers
- Some features showed high variance and wide distribution
- Missing categorical values required preprocessing
